# Stock Price vs Revenue Analysis

This project analyzes the relationship between stock price movements and company revenue using Tesla and GameStop as case studies. The objective is to compare market performance with fundamental financial growth.

---

## Project Overview

The goal of this project is to explore whether stock price trends reflect actual company revenue growth. By combining financial data extraction, web scraping, and visualization, this project highlights the relationship between fundamentals and market behavior.

---

## Companies Analyzed

- Tesla (TSLA)
- GameStop (GME)

---

## Project Workflow

### 1. Data Collection
- Extracted historical stock price data using the yfinance API  
- Scraped revenue data from financial sources using BeautifulSoup  
- Used Requests to retrieve webpage data  

### 2. Data Cleaning and Preparation
- Cleaned and formatted financial datasets using Pandas  
- Converted date columns to appropriate formats  
- Removed missing or inconsistent values  

### 3. Data Visualization
- Created time-series plots comparing stock price and revenue  
- Analyzed revenue growth trends alongside stock performance  
- Visualized differences in fundamental growth versus market speculation
